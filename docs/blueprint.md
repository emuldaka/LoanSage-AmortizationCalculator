# **App Name**: LoanSage

## Core Features:

- Loan Input: Enter loan details (principal, interest rate, loan term) via input fields.
- Amortization Schedule Generation: Automatically generate an amortization schedule based on the input loan details. Calculates monthly payment, interest paid, and principal paid for each period.
- Payment Schedule: Tool that calculates accelerated payments with varying periods. Allows extra monthly payments to principal.
- Visualization: Show data by graphs (via Recharts library). Provide visual representation of loan amortization.
- Download Report: Download a PDF copy of the amortization report.

## Style Guidelines:

- Primary color: Calming blue (#468B97) to inspire confidence and stability in financial matters.
- Background color: Very light cyan (#E1F0F4), near white to allow other page elements to pop.
- Accent color: Complementary light brown (#D3CEDF), for important interactions and highlights.
- Font: 'Inter', a grotesque-style sans-serif providing a modern, neutral look, suitable for both headlines and body text.
